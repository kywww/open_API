# 교보문고 클론 서점 웹사이트

본 프로젝트는 **교보문고 웹사이트를 참고한 클론 프로젝트**로,  
프론트엔드 UI 구조 설계와 화면 구현을 통해 사용자 인터페이스를 구성하고,  
Open API 기반의 비동기 통신을 통해 데이터를 조회한 뒤 UI에 동적으로 렌더링하는 기능을 구현했습니다.  

---

## 🎯 프로젝트 목표

- 실제 서비스와 유사한 프론트엔드 구조를 구현하며  
  API 연동 및 사용자 UI 흐름에 대한 이해를 높이는 것

---

## 🧩 구현 내용 및 성과

- 교보문고 UI와 사용자 흐름을 분석하여 화면 구성과 인터랙션을 유사하게 구현
- **Kakao 책 검색 Open API** 연동
- `fetch` 기반 비동기 데이터 요청 및 처리 경험
- 외부 API 데이터 요청 → 가공 → 화면 렌더링까지의  
  프론트엔드 개발 전반적인 흐름 이해

---
